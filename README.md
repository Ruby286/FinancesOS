# FinanceOS — Programa Financiero Personal

Aplicación web de gestión financiera personal desarrollada como proyecto de portfolio. Originalmente concebida y programada en **Java**, esta versión traslada toda la lógica al navegador mediante HTML, CSS y JavaScript puro, sin frameworks ni dependencias externas.

---

## 🖥️ Demo

> Puedes ver el proyecto en vivo en: https://ruby286.github.io/FinancesOS/

---

## 📌 Funcionalidades

### Módulo 1 — Análisis de Gastos
- Introducción manual de gastos fijos mensuales (nombre e importe personalizados)
- Registro de gastos variables, compras, imprevistos y ocio
- Cálculo automático del ahorro neto mensual
- Resumen visual con barra de progreso de ahorro

### Módulo 2 — Rentabilidad de Inversiones
- Cálculo de rentabilidad a partir de capital invertido y porcentaje anual
- Proyección en días, meses o años
- Muestra rentabilidad diaria, mensual y anual estimada

### Módulo 3 — Interés Compuesto
- Proyección de capital con aportaciones periódicas (diarias, mensuales o anuales)
- Tabla completa período a período
- Desglose entre capital inicial, total aportado e intereses generados

## 🚀 Cómo ejecutarlo localmente

No requiere instalación ni servidor. Simplemente clona el repositorio y abre `index.html` en tu navegador:

```bash
git clone https://github.com/Ruby286/FinancesOS
cd financeos
# Abre index.html en tu navegador
```

## 📐 Origen del proyecto

Este proyecto nació como un programa de consola en **Java**, con el que practicaba gestión de datos financieros personales: cálculo de ahorro, análisis de inversiones e interés compuesto con aportaciones periódicas.

La versión web mantiene exactamente la misma lógica y fórmulas del programa original, trasladadas a JavaScript para hacerlas accesibles desde cualquier navegador sin necesidad de compilar ni ejecutar código.
